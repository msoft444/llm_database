# 5.7.1. USB device descriptor

5.7.1. USB device descriptor

The USB device descriptor includes the following 16-bit values:

• VID (default 0x2e8a)
• PID (default 0x000f)
• BCD_DEVICE (default 0x0100)
• LANG_ID (default 0x0409)
