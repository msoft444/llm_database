# 5.7.7. SCSI Inquiry

5.7.7. SCSI Inquiry

Returned via the SCSI Inquiry command:

• VENDOR (default "RPI", max-length 8 ASCII chars)
• PRODUCT (default "RP2350", max-length 16 ASCII chars)
• VERSION (default "1", max-length 4 ASCII chars)
